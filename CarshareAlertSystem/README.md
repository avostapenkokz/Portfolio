# Оценка риска ДТП

## Описание проекта

Создание системы оценки риска ДТП при бронировании автомобиля для поездки по определенному маршруту. 

## Данные

В наличии были следующие таблицы SQL c историческими даннными по ДТП:
- **collisions** — общая информация о ДТП
- **parties** — информация об участниках ДТП
- **vehicles** — информация о пострадавших машинах

## Используемый стек

Python, SQL, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, PyTorch, LightGBM, CatBoost

## Общий вывод

Был проведен анализ данных о ДТП и возможности создания эффективной модели оценки риска ДТП на основе указанных данных с использованием различных решений. Более подробные выводы - в рабочей третради.
