# Выявление токсичных комментариев

## Описание проекта

Разработка алгоритма определения уровня английского языка по шкале CEFR в фильмах по файлам с субтитрами для языковой школы. 

## Данные

В наличии был 241 файл с субтитрами, а также файл с названиями фильмов и разметками об уровне языка в них.

## Используемый стек

Python, Pandas, NumPy, Scikit-learn, LightGBM, NLTK, SpaCy, PyTorch, BERT, re, Matplotlib, Seaborn

## Общий вывод

После создания новых признаков из файлов с субтитрами удалось добиться значения метрики balanced accuracy 83 даже без векторизации текста.
