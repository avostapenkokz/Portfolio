# Поиск изображений по описанию

## Описание проекта

Разработка алгоритма поиска изображений по текстовому описанию. 

## Данные

В наличии были следующие данные:
- две папки с изображениями
- файлы наименованиями файлов с изображениями, идентификаторами запросов и текстами запросов для поиска исзображений
- файлы с оценками соответствия запросов и изображений, полученные от пользователей и экспертов

## Используемый стек

Python, Pandas, NumPy, Scikit-learn, PyTorch, BERT, CLIP

## Общий вывод

После векторизации текстов и изображений и экспериментов с различными алгоритмами задача была выполнена при помощи развертывания нейросети CLIP.
