# Определение возраста покупателей

## Описание проекта

Требуется создать нейросеть для определения возраста покупателей на основе фотографий.

## Данные

В вашем распоряжении одна папка со всеми изображениями и CSV-файл labels.csv с двумя колонками: file_name и real_age.

## Используемый стек

Python, Keras, Matplotlib

## Общий вывод

- Добиться требуемого показателя метрики MAE удалось после масштабирования значений пикселей изображений при помощи нейронной сети ResNet-50, предобученной на наборе данных ImageNet, c подстройкой под параметры задачи регрессии (один нейрон в выходном слое и функция активации ReLu).
- Значение метрики удалось улучшить после добавления аугментации с отражением горизонтали.
- Наилучшего значения метрики удалось добиться после подбора оптимального количества эпох.
