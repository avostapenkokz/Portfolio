# Прогнозирование температуры звезд

## Описание проекта

Требуется создать нейросеть для предсказания температуры на поверхности звезд на основе косвенных данных.

## Данные

В наличии были следующие данные:
- Относительная светимость L/Lo — светимость звезды относительно Солнца
- Относительный радиус R/Ro — радиус звезды относительно радиуса Солнца
- Абсолютная звёздная величина Mv — физическая величина, характеризующая блеск звезды
- Звёздный цвет (white, red, blue, yellow, yellow-orange и др.) — цвет звезды, который определяют на основе спектрального анализа
- Абсолютная температура T(K) — температура на поверхности звезды в Кельвинах
- Тип звезды

## Используемый стек

Python, Pandas, NumPy, Scikit-learn, Matplotlib, PyTorch

## Общий вывод

После предобработки данных и создания дополнительных признаков удалось построить нейросеть, предсказывающую температуру звезд с метрикой RMSE почти в полтора раза ниже, чем требовалось по условиям проекта.
